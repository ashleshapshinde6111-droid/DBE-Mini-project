# DBE-Mini-project
A Bus Pass Management System designed to digitalize the process of issuing, renewing, and managing bus passes for students, employees, and general passengers.
This project helps reduce manual work, improves accuracy, and offers a user-friendly interface for both administrators and users.
Add, update, and delete user details
Approve or reject bus pass applications
Generate and manage bus pass records
View all users and issued passes
Manage routes and bus details
